# 3)Работа с git 
### Скачать проект:
```bash 
git clone <URL>
``` 
### Учесть все файлы:
```bash
git add.
```
### Сохранить коммит:
```bash
git commit -m "..."
```
### Отправка на сервер (отправка в облако):
```bash
git push
```
### Принятие в облако
```bash
git pull
```
